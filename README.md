# Global Trainings redesign

Global Trainings uchun psixologik-professional, ochiq va responsive landing page. Sahifa Ustoz, kurslar, jadval va Telegram arizasini asosiy foydalanuvchi yoʻli sifatida birlashtiradi.

## Ishga tushirish

Loyiha tashqi build jarayonisiz ishlaydi. `index.html` faylini brauzerda oching yoki oddiy statik serverdan foydalaning:

```bash
npx serve .
```

## Tuzilma

- `index.html` — sahifa kontenti va semantik tuzilma
- `styles.css` — atmosferik vizual tizim, responsive layout va animatsiyalar
- `script.js` — parallax, scroll reveal, aksesibiliti holatlari, kurslar accordioni, metodika, FAQ, mobil menyu va Telegram arizasi

## Kontent eslatmasi

Kurs nomlari, SHIFO davomiyligi, faoliyat yili va aloqa havolalari `globaltrainings.uz` sahifalaridan olingan. Jadval sanalari uydirilmaydi — foydalanuvchi rasmiy jadval sahifasiga oʻtadi. Mustaqil tasdiqlanmagan qatnashchi/natija statistikasi ishlatilmagan. Tibbiy da’volar marketing matniga aylantirilmagan va ta’limiy dasturlar uchun aniq cheklov berilgan.

Dilshod Mannopovning tasdiqlangan portreti va trening mediafayllari taqdim etilgach, hozirgi monogram karta haqiqiy vizuallar bilan almashtiriladi.
